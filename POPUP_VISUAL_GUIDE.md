## Button Behavior Matrix

| Button | Click | Effect | Storage | Window | Sound |
|--------|-------|--------|---------|--------|-------|
| **Dismiss** | Click | Stops immediately | Removes | Closes | Stops |
| **Snooze** | Click | Delays 5 min | Creates new | Closes | Stops |
| **Cancel** | Click | Removes alarm | Deletes | Closes | Stops |

## Time Format Examples

### Setting an Alarm
```
Input: 07:30
?
Stored as: {hour: 7, minute: 30}
?
Displayed as: "07:30"
```

### Countdown Display
```
If > 1 hour: HH:MM:SS (e.g., "01:45:30 remaining")
If < 1 hour: MM:SS (e.g., "45:30 remaining")
Last minute: SS (e.g., "45 seconds left")
When triggered: "Time's up!"
```

## Color Scheme

```
Normal State:
- Background: #202124 (dark gray)
- Text: #e8eaed (light gray)
- Accent: #0b93f6 (blue)
- Buttons: #303134 (darker gray)

Ringing State:
- Progress ring: #ff4f4f (red)
- Pulse: #ff9f4f (orange)
- Background: Same (dark gray)

Light Theme (Optional):
- Background: #f0f0f0 (white)
- Text: #202124 (dark)
```
